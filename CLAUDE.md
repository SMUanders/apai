# APAI

APAI er et personligt AI-drevet styresystem.

Formål:
- ekstern hukommelse
- AI-sortering af rå input
- prioriteringshjælp
- mindre mental støj
- hjælp i overgangssituationer

Produktregler:
- brugeren må ikke vælge kategori ved capture
- input kommer ind råt
- AI klassificerer bagefter
- V1 skal være simpel og robust
- ingen overengineering
- fokus er capture -> klassifikation -> næste handling -> visning

MVP:
- inbox for rå input
- AI-klassifikation
- objekter: task, project, note, waiting, event
- Today view
- Projects view
- Waiting view
- Review view
- manuel korrektion af AI-output
- logging af AI-runs

Teknisk default:
- Next.js
- TypeScript
- Prisma
- Supabase Postgres
- Tailwind

Arbejdsstil:
- svar på dansk
- vær kort
- ét skridt ad gangen
- tag default-valg selv
- stop kun ved reel blocker
